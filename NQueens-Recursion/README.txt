Compile: make
Run: java -jar NQueens.jar <input file> <output file>
==============================
Purpose:
Given chessboard size n and the position of the first queen,
place n Queens on a nxn chessboard such that
no 2 queens can attack each other.
==============================
Input file format:
chessboardSize 1stQueenPositionX 1stQueenPositionY
==============================
Output file format:
1stQueenPositionX 1stQueenPositionY 2ndQueenPositionX 2ndQueenPositionY ...
